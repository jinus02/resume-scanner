# 이력서 스캐너

이력서에 첨부하는 이미지(수상경력·자격증·수료증 등)에서 텍스트를 추출해
복사·붙여넣기 편리한 형태로 보여주는 **100% 클라이언트 OCR 도구**.

## 핵심

- **무서버 / 무 API 키**: 모든 처리는 브라우저 안에서만 실행
- **Tesseract.js 5.x**: 한국어 + 영어 동시 인식 (`kor+eng`)
- **PDF.js 3.x**: PDF 파일 지원 — 디지털 PDF 는 텍스트 레이어 직접 추출(즉시·100% 정확), 스캔본은 페이지별 OCR 폴백
- **드래그 앤 드롭**: 이미지·PDF 여러 파일 동시 업로드
- **자동 필드 추출**: 발급일·발급기관·자격번호·성명·생년월일·점수 정규식 매칭
- **원클릭 복사**: 전체 텍스트 + 개별 필드 각각 복사 버튼

## 사용

```
index.html 을 브라우저로 열거나
GitHub Pages / Netlify / Vercel 정적 호스팅에 업로드
```

## 보안

- 외부 API·서버 통신 없음 (Tesseract worker·언어 데이터만 jsdelivr CDN 에서 로드)
- 업로드한 이미지는 메모리에서만 사용되며 어디에도 저장되지 않음
- API 키·시크릿 일체 포함하지 않음
- `Referrer-Policy: no-referrer` 메타 설정

## 기술 스택

| 영역 | 사용 기술 |
|------|-----------|
| OCR | Tesseract.js 5.1.1 (CDN) |
| 폰트 | Pretendard (CDN) |
| 빌드 | 없음 — 순수 HTML / CSS / JS |
| 호스팅 | GitHub Pages |
